    MY COMPILATION OF ACTIVITY

    Event Driven Programming is a way computers work by reacting to events, when you click a button or type something.
    Instead of doing things step by step, it waits for specific actions to occur. Like having a computer that only does things when
    you tell it to or when certain things happen. By so, when you interact with a program, like clicking a link, it triggers actions in response.
    This programming style makes software more interactive and responsive to user input.
